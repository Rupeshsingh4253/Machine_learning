**Q1: Scatter Plot for Age and Fare in Titanic Dataset**
```python
import seaborn as sns
import plotly.express as px

# Load Titanic dataset
titanic = sns.load_dataset('titanic')

# Plot scatter plot using Plotly express
fig = px.scatter(titanic, x='age', y='fare', title='Scatter Plot: Age vs Fare')
fig.show()
```

**Q3: Histogram with Color and Shape in Tips Dataset**
```python
import plotly.express as px

# Load Tips dataset
tips = px.data.tips()

# Plot histogram using Plotly express
fig = px.histogram(tips, x='sex', y='total_bill', color='day', pattern_shape='smoker',
                   title='Histogram: Total Bill by Sex with Smoker Pattern and Day Color')
fig.show()
```

**Q5: Distplot using Plotly express**
```python
import plotly.express as px

# Generate random data for demonstration
import numpy as np
data = np.random.randn(1000)

# Plot distplot using Plotly express
fig = px.histogram(data, nbins=30, marginal="rug", title='Distplot: Random Data')
fig.show()
```

Note: Distplot is a deprecated function in Plotly Express. Instead, you can use `px.histogram` as shown in the example.
